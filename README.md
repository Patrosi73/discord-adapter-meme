# Discord -> Fluxer Adapter

A very experimental, hacky and not so serious project that allows connecting to Fluxer using a Discord client.
This is a fork intended for [Fluxtop](https://github.com/Patrosi73/Fluxtop) which improves upon the original code. For more information on the fork itself, see [Fluxtop's README](https://github.com/Patrosi73/Fluxtop?tab=readme-ov-file#changes-from-upstream-discord-adapter-meme).
